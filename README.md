YouTube Clone

Project Overview

This project is a YouTube UI Clone developed using HTML and CSS. The website replicates the basic layout of YouTube, including a navigation bar, sidebar, search section, and video cards.

Technologies Used

- HTML5
- CSS3

Features

- Responsive Design
- Navigation Bar
- Search Bar
- Sidebar Menu
- Video Thumbnail Cards
- Mobile-Friendly Layout
